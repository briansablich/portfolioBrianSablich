PROYECTOS

ASP.NET + C# + SQL + BOOTSTRAP + VISUAL STUDIO + GITHUB

Desarrollada en equipo de tres personas, en poco más de un mes a través de GitHub.
El sistema está compuesto por dos tipo de usuarios: administradores y choferes, con sus validaciones de seguridad. El de administradores permite el ABM de viajes, direcciones, vehiculos, choferes y clientes, con relaciones entre sí. El de choferes solo permite ver sus viajes, con ubicaciones georreferenciadas y distintos listados. También tiene distintos filtros de búsqueda en todas las categorias.
